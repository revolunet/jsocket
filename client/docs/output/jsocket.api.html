jsocket API

Chaque callback est appele avec en parametres un dictionnaire comprenant les attributs suivants:

  • param.value: value JSON field (empty string '' if not exists)
  • param.app: app JSON field (empty string '' if not exists)
  • param.channel: channel JSON field (empty string '' if not exists)

Exemple:

jsocket.api.app['myapp'].onJoin(args) {
  args.value = true;
  args.app = 'myapp';
  args.channel = 'myapp_channel';
};


This class is a singleton and cannot be created directly.

Parametres de configuration:

  • tcp.host: Host pour le core jsocket.core.tcp.loaded
  • tcp.port: Port pour le core jsocket.core.tcp.loaded
  • http.url: Url pour le core jsocket.core.http.loaded
  • http.refreshTimer: Temps entre chaque rafraichissement (en ms) pour le core jsocket.core.http.loaded
  • websocket.host: Host pour le core jsocket.core.websocket.loaded
  • websocket.port: Port pour le core jsocket.core.websocket.loaded

Ce parametre peut etre changer directement comme dans l'exemple ci-dessous ou via la methode jsocket.api.configure

Exemple de configuration:

jsocket.api.settings = {
  tcp: {
    host: 'localhost',
	port: 8080
  },
  http: {
    url: 'http://localhost:8081/',
	refreshTimer: 1000
  },
  websocket: {
    host: 'localhost',
    port: 8082
  },
  vhost:'test.quickprez.com'
};
