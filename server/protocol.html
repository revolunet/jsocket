Protocol::

Connected | Auth | Create | Remove | ChanAuth | Join | Part | Nick | Message | Forward

Connection au serveur

{"cmd" : "connected", "args": "null"}
Code retour::
{"from": "connected", "value": "HASH_UNIQUE_DE_CONNECTION"}

{"from": "auth", "value": true}
{"from": "auth", "value": false}
